A Few Accomplishments

See below projects completed by Jelissa Whatley utiliziing various full stack development technologys. These technologies includes Javscript, HTML, CSS, Jquery, Es6, Node.js, Git, PostgresSQL, Heroku and Netlify.

Foodie

Foodie is a backend focused application that was created to allow local foodies to search different restaurants in the metro atlanta area. This application was built with Javascript, Express, Sequelize, and EJS. Foodie is hosted on Heroku using Heroku CLI. This app not only allows the user to swipe left or right on restaurants in the Atlanta area but lets the user check out the menu, popular dishes, hours of operation, ect. by clicking the image. Once the user finds a restaurant they'd like to go to they'd like it and the restaurant gets added to their favorites list.

Barrel Hop

Barrel Hop is a brewery search app created to allow users to do a quick search of breweries in their local area and get an idea of weather in that area. This project is built with Javascript, Express and hosted on Heroku using Heroku CLI. The search is done utilizing multiple fetches throughout the app. .ENV is utilized in a node-fetch in an express route to ping an API that requires a API key. This data is then fetched within the static JS file.

Weather App

Weather App is a weather search application utlizing Open Weather Map API to query data, generate the weather and set the background to a picture of the city using Unsplash API. Cards containers are generated using JQuery, APIs are queried with fetch request corresponding to the zip code entered, eerror handlers are in place to handle zip codes that do not return data. Image is set using a fetch with a fetc.

Tic Tac Toe

Tic Tac Toe game app is a fully functioning two player tic tac toe game. Game board is generated using nested for loops and pure vanillia JavaScript. It has full functinality includeing the following: score tracking, tie checks, and reset button to clear board.
